# Architecture Decision Records

Cada ADR documenta uma decisão importante de arquitetura.

## Formato

```markdown
# ADR-NNNN: Título

## Contexto
Problema que motivou a decisão.

## Decisão
O que foi decidido e por quê.

## Consequências
Impactos positivos e negativos.

## Alternativas Consideradas
Outras opções e motivo da rejeição.
```
